# Background Clash – v2 Roadmap

Documento guía para la **versión 2 de Background Clash**.
Este archivo es la **fuente de verdad** para trabajar con IA y acelerar el desarrollo sin perder enfoque.

Objetivo de v2: pasar de un generador simple de fondos a una **biblioteca creativa rápida, usable y reutilizable**, pensada para diseñadores y devs.

---

## PRINCIPIOS DEL PROYECTO

* Menos controles, mejores resultados
* Todo debe ser **copy–paste friendly**
* El preview es el centro del producto
* UX inmediata: cero curva de aprendizaje
* Cada fondo debe ser:

  * Visualmente usable
  * Editable
  * Reproducible

---

## ESTADO ACTUAL (MVP v1 – COMPLETADO)

### Funcionalidades

* Biblioteca de backgrounds predefinidos
* Selección visual por cards
* Preview en tiempo real
* Copiar CSS del background
* Gradientes suaves y limpios

### UI

* Sidebar con backgrounds
* Área de preview principal

---

## VERSION 2 – ALCANCE GENERAL

Background Clash v2 introduce:

* Control creativo ligero
* Variaciones dinámicas
* Organización avanzada
* Export y reutilización mejorados

---

## FASE 1 – UX / UI POLISH

### Objetivo

Hacer que la herramienta se sienta más sólida, moderna y profesional.

### Tareas

* Mejorar jerarquía visual del sidebar
* Estados claros:

  * Selected
  * Hover
  * Active
* Preview más limpio (menos texto, más foco visual)
* Botón Copy CSS con feedback visual
* Animaciones suaves entre backgrounds
* Modo oscuro

### Resultado esperado

Sensación de herramienta cuidada y lista para uso diario.

---

## FASE 2 – CONTROLES DE VARIACIÓN

### Objetivo

Permitir personalización sin romper la simplicidad.

### Controles

* Angle del gradiente
* Intensidad del gradiente
* Saturación
* Luminosidad
* Opacidad global

### Reglas

* Controles opcionales
* Valores seguros por defecto

---

## FASE 3 – VARIANTS & RANDOM

### Objetivo

Exploración rápida y creatividad sin fricción.

### Funcionalidades

* Botón Generate variant
* Botón Randomize
* Seed reproducible
* Variaciones basadas en:

  * Colores
  * Ángulo
  * Intensidad

---

## FASE 4 – ORGANIZACIÓN & DESCUBRIMIENTO

### Objetivo

Escalar la librería sin perder orden.

### Funcionalidades

* Categorías:

  * Soft
  * Dark
  * Neon
  * Pastel
  * Nature
* Filtro por tipo
* Búsqueda por nombre
* Favoritos ⭐

---

## FASE 5 – EXPORT & USABILIDAD

### Objetivo

Hacer los fondos reutilizables en múltiples contextos.

### Export

* Copy CSS (default)
* Export como:

  * CSS variable
  * Tailwind config snippet
  * Inline style
* Mostrar compatibilidad (CSS support)

---

## FASE 6 – PRESETS & COLLECTIONS

### Objetivo

Flujos rápidos y reutilizables.

### Funcionalidades

* Colecciones predefinidas
* Guardar fondos personalizados
* Persistencia en localStorage

---

## FASE 7 – SHARE & DEV FEATURES (OPCIONAL)

### Share

* URL con parámetros del fondo
* Fondo reproducible por link

### Dev Mode

* Mostrar código CSS completo
* Toggle variables / raw values

---

## CRITERIOS DE CALIDAD

* CSS limpio y moderno
* Transiciones suaves
* Preview fiel al resultado final
* UI consistente

---

## DEFINICIÓN DE ÉXITO v2

Background Clash v2 es exitoso si:

* Permite elegir un fondo en < 10 segundos
* El CSS se usa sin modificaciones
* Se convierte en herramienta recurrente

---

## NOTAS PARA LA IA

* No sobrecargar la UI
* Priorizar presets y variantes
* Mantener consistencia visual
* Cada feature debe justificar su existencia
